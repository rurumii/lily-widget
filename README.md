# しなもん

<br>

## 概要 · about

a minimal macOS time widget built with SwiftUI

<br>
## 詳細 · details
```
platform   macOS 15.7+
language   Swift / SwiftUI / WidgetKit
font       Didot
```

<br>

## 使い方 · install

1. clone the repo
2. open `widget.xcodeproj` in Xcode
3. **Product → Archive → Custom → Copy App**
4. move `.app` to Applications and launch

<br>

## 備考 · notes

- widget name: `しなもん` (cinnamon)
- custom background image supported
- no dependencies
